invert() は CSS の関数で、入力画像の色サンプルを反転します。結果は {{cssxref("<filter-function>")}} です。

構文

invert(amount)

引数

amount
変換の度合いで、 {{cssxref("<number>")}} または {{cssxref("<percentage>")}} で指定します。 100% の値では完全に反転され、 0% の値では入力が変更されないままになります。 0% と 100% の間の値は、効果の割合です。補間時の欠損値は 0 です。

例

invert(0)     /* 効果なし */
invert(.6)    /* 60% 反転 */
invert(100%)  /* 完全に反転 */
