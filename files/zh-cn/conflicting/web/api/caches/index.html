{{APIRef("Service Workers API")}}

caches 是一个 {{domxref("ServiceWorkerGlobalScope")}} 接口的只读属性，返回与当前service worker相关联的{{domxref("CacheStorage")}}对象。

语法

var myCacheStorage = self.caches;

返回值

一个 {{domxref("CacheStorage")}} 对象。
